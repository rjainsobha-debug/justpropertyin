# JustProperty.in starter website

Files included:
- `index.html`
- `properties.html`
- `localities.html`
- `services.html`
- `about.html`
- `contact.html`
- `privacy-policy.html`
- `styles.css`
- `script.js`
- `assets/logo.svg`

## Deployment
Upload all files to the root of your GitHub repository and enable GitHub Pages or deploy via Cloudflare Pages / Netlify / Vercel.

## Notes
- This starter pack uses a `mailto:` lead form for simple static deployment.
- Replace demo listings with verified inventory before running ads.
- Review the privacy policy with a legal advisor.
